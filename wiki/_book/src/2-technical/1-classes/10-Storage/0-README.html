Storage

کلاس storage سورس

این کلاس برای مدیریت داده های برنامه که در نظر داریم بر روی localStorage کاربر بنویسیم یا بخوانیم ساخته شده است.

در این کلاس، از یک کلاس کمکی به نام AesJs استفاده شده است که این امکان را به ما میدهد که داده های رمزنگاری شده متقارن در localStorage ذخیره کنیم و بخوانیم با این حالت امنیت در سطح قابل قبولی برای برنامه تخمین می گردد. (این کار با متدهای دارای پسوند Encrypted انجام می شود).

متد get سورس

برای دریافت مقدار ذخیره شده با ارائه کلید آن مقدار که امکان ارائه و مقدار پیش فرض برای جلوگیری از خطا را نیز دارد.

متد getInt سورس

این متد معادل get است با این تفاوت که مقدار بازگشتی int می باشد.

متد getObject سورس

مشابه get و با این تفاوت که object بر میگرداند.

متد set سورس

این متد با دریافت یک کلید و یک مقدار، مقدار مورد نظر را در کلید ارائه شده بر روی localStorage می نویسد.

متد setObject سورس

معادل set با این تفاوت که object را به صورت json ذخیره می کند.

متد setEncrypted سورس

یک مقدار را در قالب کلید و با اعمال کلید رمزنگاری AES ذخیره می کند و خود کلید هم در برنامه هر بار پردازش می شود.

متد getEncrypted سورس

دریافت یک مقدار encode شده ذخیره شده در localStorage را فراهم میکند.
